# ver.epsilon 상세 리드미

## 1) 버전 개요
- **핵심 목표**: SNN 규칙을 코드로 정리하고 감정 분류기를 분리.
- **중심 접근**: Exc/Inh/Reg 뉴런 규칙 명시 + 네트워크 데모 + 분류기 모듈화.

## 2) 핵심 파일 구성
- `ver.epsilon/main.py`
  - SNN 규칙(입력 결합, 메모리, 가중 평균, 조절 뉴런 등)을 체계적으로 구현.
- `ver.epsilon/emovec_autorun_cls.py`
  - 감정 코드 분류 + 4D 벡터 매핑 모델 학습 및 리포트 저장.

## 3) 핵심 아이디어 정리
- SNN의 동작 규칙을 **명시적 코드 구조**로 정리해 디버깅/튜닝 가능.
- 감정 벡터 분류기를 별도 모듈로 분리해 **교체 가능성 확보**.

## 4) 이전 버전 대비 개선/특징
- gamma의 프로토타입에서 **규칙적/명시적 SNN 구현**으로 발전.
- 감정 분류기를 분리해 **학습 파이프라인 독립성** 확보.

## 5) 실행 방법 (처음 보는 사람용)
- **SNN 데모 실행**
  ```bash
  python ver.epsilon/main.py
  ```
- **감정 분류기 학습 실행**
  ```bash
  python ver.epsilon/emovec_autorun_cls.py
  ```

## 6) 한계 및 개선 아이디어
- `ipt_to_vec4_stub`은 임시 난수 → **분류기 연동 필수**.
- 조절 뉴런 효과(임계값/드롭아웃/NE 토글)를 **정량 평가**해야 함.

## 7) 다음 단계 제안
- 분류기 출력 벡터를 SNN 입력으로 직접 연결.
- 도착 조건/종료 조건을 보다 안정적으로 정의.
