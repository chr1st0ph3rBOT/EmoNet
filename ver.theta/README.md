# ver.theta

최신 설계(고정 K, NTL 기반 ΔW, 로그/시각화/GUI 포함)를 반영한 실행 가능한 프로토타입입니다.

## 핵심 설계 요약
- **NTL = [D, S, NE, M]**
  - 도파민(D), 세로토닌(S), 노르에피네프린(NE), 멜라토닌(M)
- **K(막전위)**
  - **고정 범위**를 가지며 입력 가중치 선형 결합 후 **클리핑**합니다.
- **ΔW(가소성)**
  - **연결된 뉴런 수(연결성 규모)를 대변**합니다.
  - ΔW는 **NTL에 의해 조절**됩니다.
- **메모리 규칙**
  - 현재는 실험용 기본 규칙이며, 로그 기반으로 개선 예정입니다.

## 실행 방법
```bash
python ver.theta/main.py
```

## 기능
- **GUI**
  - 텍스트 입력 및 NTL 수동 입력
  - 스텝 실행/연속 실행
- **로그**
  - `ver.theta/logs/`에 실행 로그 기록
- **시각화**
  - NTL, K, 연결 수(ΔW 의미)를 시계열 그래프로 표시

## 설계 상세
### 1) K 업데이트
- 입력 뉴런들의 K와 가중치를 선형 결합하여 막전위를 계산합니다.
- 고정 범위 `[K_MIN, K_MAX]`로 클리핑합니다.

### 2) ΔW(연결성) 업데이트
- ΔW는 해당 뉴런에 연결된 **다른 뉴런들의 수**를 의미합니다.
- NTL에 따라 연결 수의 증감 폭이 조절됩니다.
- 예시 아이디어:
  - 도파민/NE가 높을수록 연결 증가 가능성↑
  - 세로토닌/멜라토닌이 높을수록 연결 감소 가능성↑

### 3) 로그/시각화
- 매 스텝마다 NTL, K, 연결 수 변화량을 기록합니다.
- GUI에 실시간 그래프로 출력합니다.

## 메모
- NTL 추출은 현재 수동 입력 방식입니다(LLM 결과 입력).
- ML 방식은 정확도가 낮아 fallback으로만 고려합니다.
